# Projects

## 1. DocuMind AI
An intelligent AI-powered document parsing agent and operations dashboard. It allows businesses to automate document ingestion, extract key data parameters within seconds, and eliminate manual paperwork drag.
- **Problem Resolved**: High amount of manual hours spent extracting information from invoices and invoices.
- **Tech Stack**: React, Node.js, AI APIs, custom secure databases.

---

## 2. Automation Pick-up Service
A comprehensive scheduling and operational workflow automation pipeline that coordinates real-time tasks, automates communication triggers, and syncs business calendars and databases automatically.
- **Problem Resolved**: Repetitive manual email follow-ups, date scheduling conflicts, and data synchronization lags.
- **Tech Stack**: Firebase, HubSpot API, Google Calendar API integrations, Node.js.

---

## 3. Gym Management Portal
A fully integrated administration and dashboard analytics portal designed to manage memberships, track attendance, and provide detailed operational overhead breakdowns for facility staff.
- **Problem Resolved**: Scattered spreadsheets, manual check-in verification, and unoptimized attendance reporting.
- **Tech Stack**: Django (Python), PostgreSQL, React, Tailwind CSS.

---

## 4. AI Portfolio Assistant
A modern, high-velocity floating chatbot assistant integrated into the developer portfolio that answers visitor questions dynamically by reading from a secure serverless markdown knowledge base.
- **Problem Resolved**: Standard portfolio pages are static and cannot engage visitors in real-time or securely proxy AI queries without exposing backend API credentials.
- **Tech Stack**: Astro 6, TypeScript, Groq API, Vercel Serverless Functions.
